# Dyna.Ai Review Rubric

Use this rubric for brand QA, visual review, and release readiness.

## Severity

- `P0`: Release blocker. Legal, brand identity, production, partner/customer, false-claim, or vendor-critical issue.
- `P1`: Major issue. Serious readability, hierarchy, source-of-truth, proof, layout, or channel-fit problem.
- `P2`: Moderate improvement. Fix before final release when feasible.
- `P3`: Minor polish. Does not block release.

## Approval Status

- `approved`: Ready for use or handoff.
- `approved with edits`: Usable after listed fixes.
- `revise and resubmit`: Needs another review before release.

## Review Order

1. Brand identity
2. Message clarity
3. Layout and hierarchy
4. Color and typography
5. Imagery and graphics
6. Data, charts, and proof
7. Channel or format fit
8. Production and export readiness

## Minimum Review Output

Return:

- Overall assessment
- Source of truth used
- Visual quality assessment
- Priority issues
- Suggested fixes
- Production risks
- Approval status
