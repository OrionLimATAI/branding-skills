# Dyna.Ai Channel Rules

Use the most specific official source for the asset. When a channel guideline conflicts with the foundation, follow the channel guideline for that channel.

## Source Priority

| Channel | Source |
|---|---|
| Visual identity | `Dyna.Ai Brand Visual Identity Guidelines .pdf` |
| Social and digital | `Dyna.Ai socialmedia Guidelines.pdf` |
| Events and booths | `Dyna.Ai Event Guideline.pdf` |
| Office and environment | `Dyna.Ai Brand Guideline - Office.pdf` |
| Presentations | Official Dyna.Ai 2026 Company Profile PPT template and `dyna-presentation-design` |

## Skill Routing

| User intent | Use |
|---|---|
| Core brand rule, source choice, cross-channel task | `dyna-brand-foundation` |
| Campaign idea, launch KV, integrated visual system | `dyna-campaign-design` |
| LinkedIn, EDM, web banner, digital ad, webinar visual, partner social | `dyna-social-digital-design` |
| Website page, landing page, webinar registration page, hero-led first screen | `dyna-website-landing-design` |
| Sales visual, onepage, solution brief, datasheet, case study, sales PDF | `dyna-sales-enablement-design` |
| PPT, deck, slides, company profile, sales deck, webinar deck | `dyna-presentation-design` |
| Dashboard, AI console, KPI view, analytics view, product UI direction | `dyna-dashboard-design-guideline` |
| Event KV, booth visual, webinar cover, billboard, vendor event spec | `dyna-event-design-guideline` |
| Reception, showroom screen, signage, ID, lanyard, bag, apparel | `dyna-office-environment-design` |
| Audit, QA, visual quality check, pre-release approval | `dyna-brand-reviewer` |

## Collaboration Rule

When a request crosses channels, name the primary skill, supporting skills, and reviewer step before producing the final output.
